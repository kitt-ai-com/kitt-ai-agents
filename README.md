## 사용 방법

### 1. Claude Code에서 사용 (개발자/CEO용)

```bash
# 프로젝트 폴더로 이동
cd ai-company-agents

# Claude Code 실행 → 자동으로 루트 CLAUDE.md 로드
claude

# 전체 조직으로 질문
> "신규 쇼핑몰 런칭 전략을 세워줘"

# 특정 팀으로 이동해서 질문
cd marketing
claude
> "메타 광고 캠페인을 기획해줘"

# 전략기획실의 풀 프로세스 실행
cd strategic-hq
claude
> "B2C SaaS 사업 진출을 검토해줘"
```

### 2. 슬랙에서 사용 (팀원용)

슬랙봇을 통해 VS Code 없이 팀 에이전트와 대화할 수 있습니다.

```
@봇 마케팅 메타 광고 캠페인 기획해줘      → 질문
@봇 mk 광고 기획해줘                     → 짧은 별칭
@봇 마케팅-학습 리뷰 카루셀이 CTR 2배     → 학습 등록
@봇 마케팅-기준 건기식은 식약처 기준 준수   → 기준 등록
```

설치 방법은 `slack-bot/README.md`를 참고하세요.

### 3. 향후 확장
- 웹 대시보드에서 팀 선택 → 대화
- 노션 연동으로 결과물 자동 저장

## 커스터마이징 가이드

### 새로운 팀/에이전트 추가
1. 새 폴더 생성: `mkdir [팀이름]`
2. `CLAUDE.md` 작성 (기존 팀 참고)
3. 루트 `CLAUDE.md`의 조직도에 추가

### 에이전트 역할 수정
- 해당 팀의 `CLAUDE.md` 파일을 수정
- 역할 정의, 행동 규칙, 프로세스를 조정

### 고객사별 커스텀
- 이 프로젝트를 복사하여 고객사별 폴더 생성
- 고객사 업종에 맞게 에이전트 구성 변경
- 고객사 SOP, 브랜드 가이드 등을 각 CLAUDE.md에 추가
